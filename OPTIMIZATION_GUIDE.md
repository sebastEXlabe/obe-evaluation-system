## 三、按优先级的优化方案

### P0 — 论文必需（马上做）

#### P0-1：毕业要求→课程目标 映射（CO↔PO Mapping）

**文献依据**：
- 林清滢等(2022) — "基于工程教育认证的毕业要求达成度评价研究与实践"
- 胡晓宏等(2020) — "工程教育认证毕业要求达成度评价机制研究"
- 吴家皋等(2020) — "基于E-R图的工程教育专业认证课程达成度评价方法研究"
- 李宏然等(2020) — "基于毕业要求达成度评价的教学过程质量监控机制"

**当前状态**：系统只有"课程目标→指标点→评价方式"的单向链路

**优化方案**：
1. 数据库新增 `graduation_requirement` 表（工程教育认证12条标准毕业要求）
2. 新增 `objective_requirement_mapping` 表（课程目标↔毕业要求映射，含权重）
3. 达成度页面新增"毕业要求达成度"Tab
4. 计算公式：`毕业要求i达成度 = Σ(课程目标j达成度 × 映射权重j→i) / Σ(映射权重j→i)`
5. 可视化：课程目标→毕业要求热力图矩阵

**工作量**：1-2天（数据库2张表 + 后端2个接口 + 前端1个Tab）

#### P0-2：达成度计算改为"最小值法"

**文献依据**：工程教育认证标准要求"所有指标点均达到最低标准"

**当前状态**：`达成度 = Σ(权重×达成度)` — 加权平均法，强项可弥补弱项

**优化方案**：
- 保留加权平均作为"整体达成度"展示
- 新增"最小值法"作为"达成度评价"依据
- 课程目标达成度 = `Min{该目标下所有指标点的达成度}`
- 当最小值 < 0.6 时，标记该目标为"未达成"

**工作量**：0.5天（公式调整 + 前端双值展示）

#### P0-3：数据导出 PDF + Excel

**竞品参考**：高校OBE系统使用 iText7 PDF + Excel，人才培养达成度平台使用 EasyExcel

**优化方案**：
| 导出类型 | 技术 | 内容 |
|---------|------|------|
| 达成度报告 PDF | Apache PDFBox（开源免费）| 课程目标达成度表 + 雷达图 + 改进建议 |
| 成绩单 Excel | EasyExcel | 学生姓名/组名/贡献系数/最终成绩 |
| 数据表格 Excel | EasyExcel | 课程目标树所有数据 |

**工作量**：1天（PDFBox模板 + EasyExcel导出）

### P1 — 重要增强（论文显著加分）

#### P1-1：完整 AHP 权重分配

**文献依据**：张磊(2026), 刘荟萃(2026), 刘乐(2025), 方丹(2024)

**当前状态**：只验证权重之和=1，无两两比较矩阵

**优化方案**：
1. 教师端新增 AHP 权重配置页面
2. 课程目标间两两比较（1-9标度法）
3. 自动计算特征向量作为权重
4. 自动计算 λmax、CI、CR 进行一致性检验（CR<0.1通过）
5. 可视化：权重柱状图

**工作量**：1天（1个新页面 + 后端AHP计算服务）

#### P1-2：多阈值预警 + 持续改进闭环

**竞品参考**：工程教育认证课程管理平台（0.7阈值+PDCA工单）

**当前状态**：改进建议是静态文字，无预警、无工单

**优化方案**：
1. 可配置预警阈值（默认0.6预警线，0.7认证线）
2. 达成度低于阈值时：前端红色高亮 + 自动生成改进工单
3. 改进工单表：`improvement_task`（目标ID、当前达成度、建议措施、负责人、截止日期、完成状态）
4. PDCA闭环：评价→预警→工单→整改→再评价→趋势对比
5. 首页仪表盘：待整改项目数、整体达成度趋势

**工作量**：1.5天（数据库1张表 + 后端工单CRUD + 前端预警+仪表盘）

#### P1-3：多学期趋势分析

**竞品参考**：高校OBE系统（趋势折线图）

**当前状态**：只有当前快照，无历史对比

**优化方案**：
1. 达成度结果关联 `semester` 字段
2. 折线图：同一课程目标跨学期对比
3. 改进效果可视化：标注改进措施实施时间点，观察达成度变化

**工作量**：0.5天（数据关联 + ECharts折线图）

### P2 — 锦上添花

#### P2-1：批量 Excel 导入

**竞品参考**：高校OBE系统

**优化方案**：
- 学生名单批量导入（下载模板→填写→上传）
- 成绩数据批量导入（替代逐一手动录入）
- 课程目标体系模板化导入

**工作量**：1天

#### P2-2：5维雷达图扩展

**当前**：3维（知识/能力/素养）

**优化**：按课程目标数量动态展开，每个目标一个维度

**工作量**：0.5天

#### P2-3：首页仪表盘

新增首页面板，显示：
- 整体达成度仪表盘
- 待整改项统计
- 本学期课程概览
- 最近操作记录

**工作量**：1天
